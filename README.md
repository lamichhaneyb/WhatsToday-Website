# Whats Today?

### Whats Today? is a simple website used to show current events that happened on a given day. It shows wikipedia articles for that day, giving a user options to save them if they choose to make an account.  

## Setting up a Project

Create a Python Virual Enviorment  

```python -m venv venv```

If you're on linux make sure to actiavte it by doing  

```source ./venv/bin/activate```

When that finishes make sure to install required libraries  

```pip install -r requirements.txt```

To start the project simply run

```flask run```
